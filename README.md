## How to run
Simply clone it, and type 'java Main' on your terminal to run.

## Terminal
    Welcome to Morse Code translator by Uno! (type ':wq' to exit)
    -> Start by entering text or morse:
 
## Input 
    1) aaron swartz
    2) july 4 1776
    3) .... . .-.. .-.. --- /  .-- --- .-. .-.. -..
    4) .--- ..- .-.. -.--/ ....-/ .---- --... --... -....
    5) ..- -. ---/ -... .- - -... .- -.-- .- .-.
  
## Output
    1) .- .- .-. --- -./ ... .-- .- .-. - --..
    2) .--- ..- .-.. -.--/ ....-/ .---- --... --... -....
